# FM Simulator

Браузерный футбольный менеджер, в котором можно выбрать реальный клуб, управлять составом, проводить сезоны, участвовать в чемпионатах и кубках, совершать трансферы и подниматься по футбольной пирамиде.

Проект развивается как полноценный симулятор карьеры менеджера: от работающего одиночного сезона к глубокой системе развития игроков, сотрудников, финансов, инфраструктуры и многосезонной карьеры.

> Проект находится в активной разработке. Текущая версия уже содержит основной играбельный цикл, но ещё не является финальным релизом.

## Возможности текущей версии

### Страны и соревнования

В игре доступны шесть национальных футбольных систем:

- Англия (Premier League, EFL Championship, EFL League One, EFL League Two);
- Германия (Bundesliga, 2. Bundesliga);
- Испания (La Liga, Segunda División);
- Италия (Serie A, Serie B);
- Франция (Ligue 1, Ligue 2);
- Россия (МИР Российская Премьер-Лига, PARI Первая лига, LEON Вторая Лига А (Золото, Серебро), LEON Вторая лига Б (Группы 1-4)).

### Режим карьеры

Уже можно:

- выбрать страну, лигу и любой доступный клуб;
- начать новую карьеру;
- просматривать главную страницу клуба;
- проводить полноценные сезоны;
- следить за календарём и результатами матчей;
- просматривать турнирные таблицы;
- участвовать в национальном кубке;
- повышаться в дивизионе;
- вылетать в нижний дивизион;
- переходить к следующему сезону;
- продолжать карьеру после перезагрузки страницы.

### Управление командой

Реализованы:

- состав клуба (известные фамилии игроков рандомно);
- выбор стартовых игроков;
- настройка расстановки (4-4-2 | 4-3-3 | 4-2-3-1 | 3-5-2 | 4-5-1);
- управление заявкой на матч;
- просмотр характеристик футболистов;
- проверка корректности стартового состава;
- автоматический подбор состава для клубов под управлением ИИ.

### Матчи

Игровой движок поддерживает:

- симуляцию матчей пользователя;
- симуляцию матчей остальных клубов;
- расчёт итогового счёта;
- события матча;
- обновление результатов;
- пересчёт турнирной таблицы;
- завершение тура и сезона.

### Трансферы и бюджет

В текущей версии доступны:

- просмотр трансферного рынка;
- покупка футболистов;
- продажа футболистов;
- изменение бюджета клуба;
- проверки ограничений перед трансфером.

### Сохранение

Состояние карьеры сохраняется локально в браузере.

Сохраняются данные клубов, составы, календарь, результаты, таблицы, трансферы и прогресс текущего сезона.

## Основной игровой цикл

```text
Выбор страны, лиги и клуба
        ↓
Настройка стартового состава
        ↓
Подготовка к следующему матчу
        ↓
Симуляция игрового дня
        ↓
Просмотр результата и таблицы
        ↓
Трансферы и изменение состава
        ↓
Завершение сезона
        ↓
Повышение, вылет или сохранение места
        ↓
Новый сезон
```

## Основные страницы

| Страница    | Назначение                                             |
| ----------- | ------------------------------------------------------ |
| Выбор клуба | Выбор страны, соревнования и команды для новой карьеры |
| Dashboard   | Основная информация о клубе и текущем сезоне           |
| Состав      | Управление игроками, стартовым составом и расстановкой |
| Лига        | Турнирная таблица и положение команд                   |
| Кубок       | Сетка и результаты национального кубка                 |
| Календарь   | Прошедшие и будущие матчи                              |
| Матч        | Симуляция и результат встречи                          |
| Трансферы   | Покупка и продажа игроков                              |

## Установка и запуск

### Bun

```bash
bun install
bun dev
```

### npm

```bash
npm install
npm run dev
```

После запуска приложение будет доступно по адресу, который выведет Vite.

## Сборка

```bash
bun run build
```

или:

```bash
npm run build
```

Команда выполняет проверку TypeScript и создаёт production-сборку.

## Тесты

```bash
bun test
```

или:

```bash
npm test
```

Тестами покрываются ключевые части игровой логики:

- генерация календаря;
- расчёт турнирной таблицы;
- симуляция матчей;
- кубковые соревнования;
- завершение сезона;
- трансферы;
- сохранение и восстановление карьеры.

## Ближайший ориентир

Наиболее логичная следующая версия:

1. тренировки;
2. физическая готовность и усталость;
3. тренерский штаб;
4. травмы и восстановление;
5. развитие рейтинга игроков;
6. более глубокое влияние состава на симуляцию матча.

Это добавит содержательные решения между турами и превратит сезон из цепочки симуляций в полноценное управление командой.

## Статус проекта

Проект разрабатывается как pet project и учебная площадка для построения сложной игровой логики на Vue 3 и TypeScript.

Основная цель — постепенно вырастить работающий футбольный симулятор, не превращая кодовую базу в один компонент на несколько тысяч строк. Такая трагедия уже случалась с человечеством. Повторять необязательно.
