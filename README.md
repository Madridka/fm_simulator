# FM Simulator

Браузерный футбольный менеджер с многосезонной карьерой. Игрок выбирает реальный клуб, управляет составом и академией, проводит матчи, участвует в чемпионате и кубке, выходит в другие дивизионы и развивает команду на протяжении многих сезонов.

> Проект находится в активной разработке. Основной игровой цикл уже работает, но баланс и отдельные системы продолжают развиваться.

## Что уже реализовано

- шесть футбольных систем: Англия, Германия, Испания, Италия, Франция и Россия;
- национальные лиги с повышением, вылетом и плей-офф;
- национальные кубки и переход между сезонами;
- параллельная симуляция игрового мира;
- выбор стартового состава, запасных, формации и тактического стиля;
- физическая готовность, форма, потенциал, возрастное развитие и завершение карьеры игроков;
- травмы, восстановление, карточки и дисквалификации;
- матчи с поминутной трансляцией, заменами, сериями пенальти и домашним преимуществом;
- статистика матча: владение, удары, xG, карточки и лучший игрок;
- сезонная статистика футболистов и таблицы лидеров;
- трансферный рынок, покупка и продажа игроков, бюджеты клубов;
- академии, ежегодные выпуски, резервные команды и перевод игроков между составами;
- локальное сохранение карьеры и миграции сохранений;
- фоновые расчёты игрового дня через Web Worker.

## Игровой цикл

```text
Выбор клуба → состав и тактика → матч → результаты и таблицы
     ↑                                      ↓
новый сезон ← трансферы и развитие ← управление клубом
```

Основные разделы игры: главная страница клуба, состав, академия, лига, кубок, календарь, матч и трансферы.

## Ближайший план

### 1. История и события

- внутриигровая почта и лента событий;
- архив сезонов, чемпионов, таблиц и рекордов;
- награды игрокам и история их карьеры.

### 2. Контракты и трансферный рынок

- сроки контрактов, продление, зарплаты и бонусы;
- трансферные окна, переговоры и входящие предложения;
- свободные агенты, аренды и трансферы между ИИ-клубами.

### 3. Управление матчем

- ручные замены и смена тактики по ходу встречи;
- прессинг, темп, ширина и линия обороны;
- более глубокие решения ИИ в зависимости от счёта.

### 4. Развитие команды

- командные и индивидуальные тренировки;
- усталость, мораль, сыгранность и риск травм;
- улучшение академии, базы и медицинской инфраструктуры.

### 5. Карьера менеджера

- сезонные задачи и доверие руководства;
- репутация, увольнения и предложения от других клубов;
- достижения и история карьеры менеджера.

### 6. Расширение игрового мира

- суперкубки и континентальные клубные турниры;
- квалификация в еврокубки и коэффициенты стран;
- динамическая репутация клубов и соревнований.

## Запуск

Требуется актуальная версия Node.js или Bun.

```bash
# Bun
bun install
bun dev

# npm
npm install
npm run dev
```

## Проверка проекта

```bash
# production-сборка с проверкой TypeScript
bun run build
# или npm run build

# тесты игровой логики
bun test
# или npm test
```

Тестами покрыты ключевые части симуляции: календарь, матчи, таблицы, кубки, переход между сезонами, трансферы, статистика игроков и сохранения.

## Технологии

Vue 3, TypeScript, Vite, Pinia, Vue Router, Vue I18n, PrimeVue, Tailwind CSS и Vitest.

## Правовая информация

Проект не связан с футбольными клубами, лигами или правообладателями их брендов. Названия и эмблемы используются исключительно в некоммерческих демонстрационных целях.
